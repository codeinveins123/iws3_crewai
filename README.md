# Система формирования плана восстановления студентов

CrewAI-based система для анализа успеваемости студента и формирования индивидуального плана восстановления.

## Возможности

- Загрузка данных об успеваемости (CSV)
- Загрузка мотивационного письма (TXT)
- Загрузка правил академической задолженности (TXT)
- Загрузка каталога сервисов помощи (TXT)
- Редактирование конфигурации агентов
- Автоматический HITL (Human-in-the-Loop) при необходимости уточнений - всегда в конце
- Формирование индивидуального плана восстановления

## Установка зависимостей

```bash
pip install crewai streamlit pandas python-dotenv
```

или взять из `requirements.txt`
